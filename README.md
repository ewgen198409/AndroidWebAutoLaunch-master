Простое Android-приложение, предназначенное для автоматического запуска веб-страниц в полноэкранном режиме при загрузке устройства. Идеально подходит для создания выделенных киосковых веб-интерфейсов или информационных панелей на Android-устройствах.

## Возможности

- **Автозапуск при загрузке**: Автоматически открывает указанную веб-страницу при запуске устройства
- **Полноэкранный режим**: Иммерсивный полноэкранный режим для беспрепятственного просмотра
- **Гибкий доступ к настройкам**: Доступ к настройкам через свайп с края или долгое нажатие
- **UPnP Аудио Рендерер**: Встроенная поддержка потокового аудио UPnP
- **Поддержка тем**: Светлая, темная и автоматическая темы
- **Многоязычность**: Поддержка английского, немецкого и русского языков
- **Параметры безопасности**: Опция доверия всем SSL-сертификатам (для разработки/тестирования)
- **Автовоспроизведение видео**: Автоматическое воспроизведение видео при загрузке страницы
- **Блокировка экрана**: Опция предотвращения таймаута экрана
- **Wake lock**: Поддерживает устройство в рабочем состоянии

## Требования

- Android 5.0 (API 21) или выше
- Разрешение на интернет
- Разрешение на завершение загрузки для автозапуска

### Сборка из исходного кода

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/ewgen198409/AndroidWebAutoLaunch-master.git
   cd AndroidWebAutoLaunch-master
   ```

2. Откройте проект в Android Studio

3. Соберите и запустите на устройстве или эмуляторе

## Использование

1. Установите и откройте приложение
2. Доступ к настройкам через свайп с любого края или долгое нажатие в любом месте страницы
3. Настройте желаемый URL
4. Включите "Запуск при старте" для автозапуска при загрузке
5. Настройте другие параметры по необходимости (тема, язык, параметры безопасности и т.д.)
